# Lab FAQ

---

## About the Lab

### Зачем это?

Личный research проект. Хочу понять, как дать AI-агентам долгосрочную память. Учусь, экспериментирую, публикую результаты.

### Это продукт? Стартап?

Нет. Это исследовательская лаборатория одного человека + AI-агенты. Нет инвесторов, клиентов, revenue targets.

### Кто работает в лаборатории?

- **Я** — исследователь, лаборант, автор
- **AI-агенты** — помогают с экспериментами, кодом, анализом

## Research

### Какие технологии используются?

| Component | Technology |
|-----------|------------|
| Vector DB | Chroma (local) |
| Memory | Mem0, A-MEM |
| Embeddings | OpenAI text-embedding-3-small |
| LLM | Claude 3.5, GPT-4o |

### Какие research questions?

1. Как организовать долгосрочную память агента?
2. Как объединить 8 слоёв контекста эффективно?
3. Как измерять качество памяти?
4. Как агенты должны делить память?

### Где публикуются результаты?

- Blog posts — еженедельно
- GitHub — код экспериментов
- arXiv — формальные paper (по мере готовности)
