## Channels

| Channel | Purpose | Cadence |
|---------|---------|---------|
| **Blog/Substack** | Long-form essays, experiment reports | Monthly |
| **GitHub** | Code, tools, experiments | Continuous |
| **Twitter/X** | Short insights, threads | Weekly |
| **arXiv** | Formal papers | Yearly (if warranted) |

---

## Content Types

### Experiment Reports
- What hypothesis, what method, what result
- Honest about failures
- Reproducible code links

### Tool Releases
- Code + documentation
- Usage examples
- Installation instructions

### Tutorials
- "How to replicate X"
- Step-by-step guides
- Beginner-friendly

### Essays
- Big picture thinking
- Philosophical takes on AI memory
- Future predictions

---

## Cadence

| Type | Target |
|------|--------|
| Weekly | 1 short insight (Twitter thread) |
| Monthly | 1 experiment report (blog) |
| Quarterly | 1 tool release (GitHub) |
| Yearly | 1 comprehensive essay or paper |

---

## Success Metrics

| Metric | Target |
|--------|--------|

| GitHub stars | Any is good |
| Engagement | Quality discussions |
| Citations | Would be amazing |
