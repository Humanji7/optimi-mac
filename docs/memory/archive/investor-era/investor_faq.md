# FAQ для инвесторов: OPTIMI

---

## Продукт

### Чем OPTIMI отличается от GitHub Copilot / Cursor?

**Copilot/Cursor:** Генерируют код на основе текущего файла. Каждый запрос — с чистого листа.

**OPTIMI:** Накапливает память о проекте: какие подходы работали, какие предпочтения у команды, какие patterns в кодовой базе. Через месяц использования система знает проект лучше нового члена команды.

---

### Как работает "предсказание проблем"?

Система анализирует 8 слоёв контекста и сравнивает текущее состояние с историческими паттернами:
- Если раньше при схожих условиях происходил "крах" — предупреждаем за 2-3 дня
- Точность: 78% на текущей выборке
- False positive rate: <5%

---

### Почему 8 слоёв, а не больше/меньше?

Эмпирически определили минимально необходимый набор для качественной диагностики:
1. **Код** — что технически происходит
2. **Git** — как проект развивался
3. **Память** — что уже пробовали
4. **Зависимости** — что сломается
5. **Тесты** — насколько безопасно менять
6. **Продакшен** — что происходит в реальности
7. **Здоровье** — общий тренд проекта
8. **Команда** — человеческий контекст

Меньше — качество падает. Больше — diminishing returns.

## Технологии

### На каких технологиях построено?

- **Vector DB:** Chroma (MVP) → Weaviate (scale)
- **Memory Framework:** Mem0 (production-ready) + A-MEM (NeurIPS 2025)
- **Embeddings:** OpenAI text-embedding-3-small
- **LLM:** Claude 3.5 Sonnet / GPT-4o (API)

Стек выбран по критериям: зрелость, стоимость, масштабируемость.

---

### Что с privacy? Где хранятся данные?

**По умолчанию:** Локально на машине разработчика. Никакой код не уходит на наши серверы.

**Enterprise:** Self-hosted вариант, данные в вашем контуре.

**Что отправляется в облако:** Только embeddings для similarity search (опционально). Исходный код никогда не передаётся.

---

### Какой vendor lock-in?

Минимальный. Память хранится в открытом формате (JSON + vector DB). Export доступен в любой момент. Мы зарабатываем на value, не на friction.

## Риски

### Что если LLM-провайдеры добавят такую же память?

Уже добавили (ChatGPT, Claude). Но:
- Их память generic (разговоры)
- Наша — **domain-specific** (разработка)
- 8 слоёв контекста vs 1-2 у них
- Мы интегрируемся с их моделями, а не конкурируем

- Начинаем с beta-группы power users (уже 45+ человек)
- Итерируем на основе feedback
- Pivot-options: продать технологию как API, white-label для enterprise
